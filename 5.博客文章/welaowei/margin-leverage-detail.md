“杠杆”是在交易中使用保证金账户的增加的购买力，可让您进行比在交易账户中拥有的资金更多的交易。它是一个比率，表示您拥有的资金与可以交易的资金之间的比例。例如，在没有使用保证金账户的情况下，如果您想交易1个标准手的USD/JPY，您需要拥有100,000美元。但是，使用杠杆100:1的情况下，您只需要在账户中存入1,000美元的1％保证金要求即可进行交易。

```php
杠杆 = 1 / 保证金要求

// 杠杆为50，表示为比率，50:1
```

# 杠杆与保证金的关系

“杠杆”和“保证金”是反相关的。交易者开仓时需要支付一定比例的保证金，而这个比例被称为保证金要求。交易者需要支付的实际金额称为所需保证金。通过支付保证金，交易者就可以获得杠杆效应，也就是用很少的资金交易很大的头寸。例如，交易者可以用2,000美元的保证金来交易10万美元的头寸大小，杠杆比率为50:1。所以，保证金要求越低，交易者能够获得的杠杆比率越高，但同时带来的风险也会增加。

# 杠杆如何影响保证金

“杠杆”是指用很少或没有资金，借入大量资金并控制它们的能力。例如，以100:1的杠杆率控制100,000美元的头寸，只需支付1,000美元。如果投资的价值上涨到101,000美元，回报率将是100％。然而，如果使用1:1的杠杆，投资的回报率只有微不足道的1％。这表明，杠杆可以加倍投资收益，但也可以加倍损失。因此，杠杆被认为是一把双刃剑。

如果使用100:1的杠杆进行外汇交易，那么需要从账户中拨出1,000美元作为“保证金”，以便控制10万美元的头寸。保证金通常以总头寸的百分比表示，如2%、1%、0.5%或0.25%。根据经纪商要求的保证金数额，可以计算出你的最大杠杆比例。例如，如果经纪商要求2%的保证金，则你的杠杆比例为50:1。

除了“保证金要求”之外，您可能会在您的交易平台上看到其他“保证金”术语。这些不同的“保证金”含义经常令人困惑，因此我们将尽力定义每个术语：

* 保证金要求：经纪商要求您开仓所需要的资金金额。它以百分数表示。

* 账户余额：这只是您的交易资金的另一个说法。这是您交易账户中所有资金的总额。

* 已用保证金：经纪商锁定的资金金额，以保持您目前的仓位开放状态。尽管这些资金仍然是您的，但在您手动关闭当前仓位或经纪商自动关闭持仓时，您无法接触这些资金。

* 可用保证金：这是您账户中可用于开设新仓位的资金。

* 保证金警告：如果您账户中的资金量无法覆盖您可能的亏损，您将收到这个提示。当您的账户权益低于已用保证金时，就会出现这种情况。如果发生保证金警告，经纪商将要求您在账户中存入更多资金。如果您不这样做，您所有或部分的持仓将由经纪商以市场价格关闭。

# 高杠杆账户如何风险管理？

高杠杆的交易平台可以让交易者控制巨大的头寸，而不需大量保证金。这就是为什么高杠杆可以让外汇交易者利润增加100、500、5000倍。然而，高杠杆也意味着风险增加，因此我们应该谨慎对待。我们应该时刻关注市场新闻，了解市场状况，以决定是否使用杠杆。即使使用最高杠杆的经纪商进行交易，当市场波动较大时，我们也不应该冒着风险进行杠杆交易。我们不该被贪婪所笼罩，而应该谨慎决策，以保证在长远中的优势和利润。

* 使用止损和止盈来保护交易头寸，避免超过自己能承受的损失。止损是一种自动关闭交易的指令，当价格达到你设定的限制时，经纪商会帮你平仓，以免你的亏损扩大。止盈也是一种自动关闭交易的指令，当价格达到你设定的目标时，经纪商会帮你锁定利润，以免你错过机会。

* 根据自己的风险偏好和交易计划来确定合适的杠杆率水平，不要使用过高的杠杆率。一般来说，你不应该在每笔交易中冒险超过1%或2%的资金。这意味着如果你有1000美元的资金，你最多只能在每笔交易中冒险10或20美元。根据这个原则，你可以计算出适合你的杠杆率水平。例如，如果你想开一个10000美元的交易头寸，并且只想冒险10美元，那么你最多只能使用1:10的杠杆率。

* 根据市场情况和新闻动态来调整杠杆率，避免在高波动性的市场中使用高杠杆率。市场波动性是指价格变化的幅度和频率。当市场波动性高时，价格可能会出现剧烈的波动，这对于使用高杠杆率的交易者来说是非常危险的。因为即使是很小的价格变动也可能导致很大的盈亏比例。因此，在高波动性的市场中，最好降低杠杆率水平，以减少风险。
